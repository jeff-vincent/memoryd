v0.2.0: quality-first synthesizer with journalistic prose output

Synthesizer prompts rewritten with two-stage architecture:
- Stage 1 (Value Gate): independently valuable information check
- Stage 2 (Rewrite): journalistic prose, direct and concrete

All three synthesizer paths (SynthesizeQA, Synthesize, SynthesizeConversation)
now output clear prose paragraphs instead of bullet lists. Each sentence
stands alone as a search result with specific identifiers.

SynthesizeQA is the mandatory quality gate for all proxy-captured content.
The model returns SKIP for procedural exchanges, accepts raw value without
question for everything else, and rewrites to maximally clear prose.

Additional changes accumulated since v0.1.3:
- Anthropic API key stored in macOS Keychain (credential package)
- Tray app: Set API Key dialog, synthesis indicator, health polling
- Synthesis mandatory gate: all bypass paths closed in proxy and API
- CLI: credentials set-api-key, credentials check
- Corpus eval benchmark harness (cmd/corpus-eval)
- Content quality scoring and rejection pipeline
- Steward improvements and test coverage

Tests: 23 synthesizer tests (11 new for QA + prompt structure)
